สิ่งที่เรียนรู้ในรายวิชา

HTML

เรียนรู้การสร้างโครงสร้างเว็บไซต์ด้วย HTML5 โดยได้ศึกษาจากในคลาสเรียน และเพิ่มเติมจากการดูวิดีโอสอนใน YouTube ช่องของคนไทย ประกอบกับการค้นคว้าจากเว็บไซต์ W3Schools แล้วนำความรู้มาทดลองเขียนโค้ดด้วยตัวเอง

CSS

เรียนรู้การจัดรูปแบบเว็บไซต์ด้วย CSS จากการเรียนในคลาสและศึกษาเพิ่มเติมจาก YouTube ของช่องคนไทยที่สอนการทำเว็บไซต์ ทำให้เข้าใจเรื่อง flexbox, grid และ responsive design นอกจากนี้ยังได้ค้นคว้าเพิ่มเติมจาก W3Schools ด้วย

JavaScript

เรียนรู้การสร้าง interactive ให้กับเว็บไซต์ด้วย JavaScript จากคลาสเรียนเป็นหลัก และศึกษาเพิ่มเติมจาก YouTube ช่องสอนเขียนโค้ดของคนไทย นำสิ่งที่ได้เรียนรู้มาทดลองเขียนและปรับแต่งตามตัวอย่างจากเว็บไซต์ W3Schools

GeoJSON และ Leaflet

เรียนรู้การทำงานกับข้อมูลแผนที่ผ่าน GeoJSON และการใช้ Leaflet จากการศึกษาในคลาส และค้นคว้าเพิ่มเติมจากเว็บไซต์ทางการของ Leaflet รวมถึงตัวอย่างจากช่อง YouTube ต่างๆ ลองทำและปรับแต่งจนสามารถสร้างแผนที่แบบ interactive ได้
